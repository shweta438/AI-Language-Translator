# 🌍 AI Language Translator

## 📌 Project Overview

The AI Language Translator is a desktop application developed using Python and Tkinter. It translates text from one language to another using the Deep Translator library. The application provides a simple and user-friendly interface for translation.

---

## ✨ Features

- 🌍 Translate text into multiple languages
- 📋 Copy translated text
- 🗑️ Clear input and output
- 🔄 Swap source and target languages
- 🔊 Text-to-Speech (Speak translated text)
- 🎨 User-friendly GUI

---

## 🛠️ Technologies Used

- Python 3.14
- Tkinter
- Deep Translator
- Pyperclip
- pyttsx3
